# S-System Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar o componente `SSystem.vue` com fidelidade ao design Figma (node 1279-291) — seção light com 4 cards de feature e barra de acento primária.

## Mapeamento de Tokens

| Figma / Hex | Token Tailwind | Uso |
|---|---|---|
| `#F2F2F2` (Gray/1) | `gray-1` | fundo da seção |
| `#CBCBCB` (Gray/6) | `gray-6` | bordas da seção e cards |
| `#111111` (Gray/12) | `gray-12` | títulos dos cards |
| `#565656` (Gray/11) | `gray-11` | corpo dos cards |
| `#E55A37` (Orange/9) | `primary` | label, barra de acento |

> ⚠️ Esta seção é **light** (fundo claro). Os aliases semânticos `text-primary`, `text-body`, etc. apontam para cores escuras do tema dark — usar os tokens brutos (`gray-12`, `gray-11`) para texto nesta seção.
