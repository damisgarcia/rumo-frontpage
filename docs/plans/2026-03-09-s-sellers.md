# SSellers Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar a seção `SSellers` com grid de 7 cards de funcionalidades para vendedores, cada um com texto e ilustração.

**Architecture:** Seção dark-themed padrão. Grid CSS de 3 colunas com `gap-8`. Cards 1–6 ocupam 1 coluna; card 7 "Produtos financeiros" ocupa 2 colunas (`col-span-2`). Ilustrações são imagens estáticas baixadas do Figma. Dados dos cards em array no `<script setup>`.

**Tech Stack:** Vue 3 / Nuxt 4 · Tailwind CSS v4 + BEM `@apply` · `<NuxtImg>` para imagens · sem dependências novas
